Welcome

This is a sample page with a themed Bulma navbar.

The navbar is fixed to the top and supports dark mode. Click the sun/moon icon to toggle themes. The "Quick Access" / "Tools" menu in the navbar end provides a dropdown panel.

Navbar items are generated based on placeholder Liquid logic. The active item should be styled.

To see this work with your actual Jekyll data, you would replace the default placeholder JSON in the Liquid `assign` tags with your `site.navbar` and `site.social_media` data structures.

For example, `page.url` would be `/` for the homepage (after removing `index.html`).
